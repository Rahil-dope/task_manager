# Quick Start Guide

## How to Use NovaTasks

### 1. Open the Application
Simply open `index.html` in any modern web browser. No installation needed!

### 2. Create Your First Task
- Click the **"+ New Task"** button (or press `n`)
- Fill in the task details:
  - **Title** (required)
  - Description
  - Category (e.g., Coding, Assignment, Reading)
  - Priority (Low, Medium, High, Critical)
  - Status (Pending, In-Process, Completed)
  - Deadline (optional)
  - Tags (comma-separated)
- Click **"Save Task"**

### 3. Organize Tasks
- **Drag & Drop**: Move tasks between columns to change status
- **Filter**: Use sidebar filters to show specific tasks
- **Search**: Type in the search box to find tasks
- **Sort**: Use the sort dropdown to organize tasks

### 4. Switch Views
- **Board View**: Kanban-style columns (default)
- **List View**: Detailed list of all tasks
- **Calendar View**: Monthly calendar with tasks (press `c`)

### 5. Recurring Tasks
- When creating/editing a task, check **"Recurring Task"**
- Select frequency (Daily, Weekly, Monthly, Yearly)
- Set interval (e.g., every 2 days)
- When completed, the next occurrence is created automatically

### 9. Keyboard Shortcuts
- `n` - Create new task
- `c` - Switch to calendar view
- `Escape` - Close modal

### 10. Sync (Optional)
- Click **"Sync"** button
- Enter your API endpoint URL for remote database sync
- Tasks will sync automatically every 5 minutes

## Tips

1. **Use Categories**: Organize tasks by project or type
2. **Set Deadlines**: Get notified about upcoming tasks
3. **Use Tags**: Add multiple tags for better searchability
4. **Color Code**: Assign colors to visually distinguish tasks
5. **Recurring Tasks**: Perfect for daily/weekly routines
6. **Calendar View**: Great for planning ahead

## Troubleshooting

**Tasks not showing?**
- Check if filters are active
- Clear search box
- Check browser console for errors

**Notifications not working?**
- Ensure tasks have deadlines
- Check browser notification permissions

**Data lost?**
- Check browser localStorage
- Try exporting regularly for backups
